    php 7.3 新特性

    duc 2018/12/7 php7.3

    php 7.3 于几天前正式推出，让我们来看一看都有什么变动。

    # is_countable RFC (opens new window)

    在计算不可数对象时，PHP 7.2添加了警告。该is_countable功能可以帮助防止此警告。

    $count = is_countable($variable) ? count($variable) : null;
    1

    # array_key_first和array_key_last rfc (opens new window)

    这两个函数基本上就是这个名字所说的。

    $array = [
    'a' => '…',
    'b' => '…',
    'c' => '…',
];

array_key_first($array); // 'a'
array_key_last($array); // 'c'

    原来RFC还提出array_value_first和array_value_last，但这些被多数人投了反对票。

    另一个想法array_first和array_last提议将返回一个元组[$key => $value]，但意见是混合的。现在我们只有两个函数来获取数组的第一个和最后一个键。

    # 灵活的Heredoc语法rfc (opens new window)

    Heredoc可以成为更大字符串的有用工具，尽管它们在过去有一个缩进的怪癖。

    // Instead of this:

$query = <<<SQL
SELECT * 
FROM `table`
WHERE `column` = true;
SQL;

// You can do this:

$query = <<<SQL
    SELECT * 
    FROM `table`
    WHERE `column` = true;
    SQL;

    # 更好的类型错误报告

    TypeErrors对于用于打印其全名的整数和布尔值，它已被更改为int和bool匹配代码中的类型提示。

    Argument 1 passed to foo() must be of the type int/bool
    1

    与PHP 7.2相比：

    Argument 1 passed to foo() must be of the type 
integer/boolean

    # 可以抛出JSON错误rfc (opens new window)

    以前，JSON解析错误是调试的麻烦。JSON函数现在接受一个额外的选项，使它们在解析错误时抛出异常。这一变化显然增加了一个新例外：JsonException。

    json_encode($data, JSON_THROW_ON_ERROR);

json_decode("invalid json", null, 512, JSON_THROW_ON_ERROR);

// Throws JsonException

    将list()其简写[]语法现在支持引用。

    # 相同的站点cookie rfc (opens new window)

    这种改变不仅增加了一个新的参数，它还改变了方式setcookie，setrawcookie并且session_set_cookie_params功能以不间断的方式工作。

    # PCRE2迁移rfc (opens new window)

    PCRE - “Perl兼容正则表达式”的缩写 - 已更新至v2。

    迁移的重点是最大向后兼容性，尽管有一些重大变化。请务必阅读RFC (opens new window)以了解它们。

    # MBString更新README (opens new window)

    MBStringPHP是处理复杂字符串 (opens new window)的方式。此模块已在此版本的PHP中收到一些更新。你可以在这里 (opens new window)阅读它。

    # 几个弃用rfc (opens new window)

    几个小东西已被弃用，因此可能会在代码中显示错误。

    • 未记录的mbstring函数别名
    • 带整数针的字符串搜索功能
    • fgetss()功能和string.strip_tags过滤器
    • 定义一个独立的assert()功能
    • FILTER_FLAG_SCHEME_REQUIRED和FILTER_FLAG_HOST_REQUIRED旗帜
    • pdo_odbc.db2_instance_name php.ini指令

    有关每个弃用的完整说明，请参阅RFC (opens new window)。
